# How To Use

```bash
curl -s --compressed https://xptsp.github.io/ppa/KEY.gpg | sudo apt-key add -
sudo curl -SsL -o /etc/apt/sources.list.d/bpiwrt_repo.list https://xptsp.github.io/ppa/ppa.list
sudo apt update
```
